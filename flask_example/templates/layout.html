Introduction:

This is an implementation for two different algorithms described on "MAXIMUM WEIGHT CYCLE PACKING IN DIRECTED GRAPHS, WITH APPLICATION TO KIDNEY EXCHANGE PROGRAMS" article.

The article points on two algorithms that solves kidney exchange problems, which can be modelled as cycle packing problems in a directed graph, involving cycles of length 2, 3, or even longer.

In the article we focus on the maximal exchange of circles of size 2 and 3 vertices, we demonstrate an approximation algorithm and an exact algorithm for this problem.
